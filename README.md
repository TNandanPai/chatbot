# Chatbot

## Overview
This project is a web-based chatbot developed using HTML, CSS, and JavaScript. It aims to provide users with an interactive conversational experience directly in their web browsers.

## Features
- **Interactive Conversations**: Engage users in dynamic dialogues.
- **User-Friendly Interface**: Clean and intuitive design for seamless interactions.
- **Responsive Design**: Ensures compatibility across various devices and screen sizes.

## Technologies Used
- **HTML**: Structures the web pages.
- **CSS**: Styles the chatbot interface.
- **JavaScript**: Implements the chatbot's interactive functionalities.

## Getting Started
To run this project locally:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/TNandanPai/chatbot.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd chatbot
   ```
3. **Open `ai.html` in Your Preferred Web Browser**.

## Project Structure
- `ai.html`: The main HTML file for the chatbot interface.
- `aistyle.css`: Contains the CSS styles for the chatbot.
- `script.js`: Contains the JavaScript code that powers the chatbot's interactions.

## Usage
- Open the `ai.html` file in a web browser.
- Interact with the chatbot by typing messages into the input field and pressing enter.
- The chatbot will respond to your messages, simulating a conversational experience.
